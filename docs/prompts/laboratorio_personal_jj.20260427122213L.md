# Laboratorio personal — JJ (Data Viz Expert, Issue #2)

## Filosofía de los dos repos

Este aporte sigue una separación intencional:

- **`P8_Data_Analysis` (este repo, equipo)**: contiene resultados,
  entregables y código reproducible. Lo que un evaluador o miembro
  del equipo necesita para entender **qué se hizo**, sin pasos
  intermedios.
- **`P8-Data-Analyst` (sandbox personal de JJ, repo público)**:
  contiene el **cómo se llegó hasta aquí** — exploraciones, prompts
  a IAs, scripts de proceso, dumps legibles de notebooks, fuentes
  editables de los entregables.

El sandbox no busca que los formadores recorran todos los archivos
manualmente. Está diseñado para que una IA pueda contextualizarse
con el proceso completo y reconstruir el razonamiento si se le pide.

Para humanos: este 
`laboratorio_personal_jj.20260427122213L.md` es el índice. Solo
se entra a los enlaces si se desea profundizar en un paso concreto.
